API使用说明

通过以下URL获取随机一言：

GET /api/hitokoto

请求参数：

  • c - 句子类型（可选，可多选），如：?c=a&c=b
  • encode - 返回格式（可选），值：json 或 text
  • min_length - 最小长度（可选）
  • max_length - 最大长度（可选）

返回格式：

{
  "status": 200,
  "message": "ok.",
  "data": [
    {
      "id": 123,
      "hitokoto": "一言内容",
      "type": "类型",
      "from": "出处",
      "from_who": "作者",
      "creator": "提交者",
      "length": 长度,
      // 其他字段...
    }
  ],
  "ts": 时间戳
}
